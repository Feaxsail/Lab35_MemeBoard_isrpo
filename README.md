## 📖 Описание работы

Создано веб-приложение **"Доска мемов" (MemeBoard)**, которое позволяет:

- ✅ Просматривать список мемов
- ✅ Добавлять новые мемы с указанием категории и рейтинга
- ✅ Удалять мемы

**Технологии:**
- **Backend** — ASP.NET Core Web API (C#)
- **Frontend** — HTML, CSS, JavaScript (fetch API)
- **Хранилище** — In-Memory (статический список)
